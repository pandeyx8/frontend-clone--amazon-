# E-Commerce Frontend Clone

A fully responsive **e-commerce frontend project** built using **HTML, CSS, and JavaScript**.  
This project focuses on implementing real-world shopping features such as product listing,
cart management, and order total calculation, purely on the frontend.

**Disclaimer:**  
This is an **educational project** created for learning and portfolio purposes only.  
It is **not affiliated with or endorsed by Amazon or any other e-commerce brand**, and it does **not collect real user data or payments**.

---

## Features
- Product listing page
- Add-to-cart functionality
- Cart quantity updates
- Order total calculation
- Persistent cart data using JavaScript
- Dynamic product loading via API integration
- Responsive UI for desktop and mobile

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

---

## What I Learned
- DOM manipulation using vanilla JavaScript
- Managing application state (products & cart)
- Working with external APIs
- Structuring a multi-file JavaScript project
- Implementing real-world e-commerce logic
- Building scalable frontend architecture
